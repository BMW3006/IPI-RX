# IPI-RX Link Tracker Pro

Dynamic link tracker with 15 features for visitor information.

## Features
- 📍 Location (City, Region, Country)
- 📱 Device Type
- 💻 Operating System
- 🌍 Browser
- 🗺️ Map Link
- 📶 Mobile Detection
- 🕒 Time
- 🌐 ISP
- 🔗 Referer
- 🌍 Language
- 🛡️ DNT
- 📊 Statistics
- 🔄 Auto Refresh
- 📱 Screen Info
- 🔐 Proxy Detection

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/IPI-RX.git
cd IPI-RX
pip install -r requirements.txt
